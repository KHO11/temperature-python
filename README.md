# Weather Data Analyzer (Python + NumPy)

# Overview

This project is a simple Python program that analyzes weekly temperature data. It uses the NumPy library to perform numerical calculations and basic data analysis.

The program stores temperature data in a NumPy array, calculates statistics such as the average temperature, and identifies temperatures that are above the average. It also demonstrates the use of loops and conditional logic to process the data.

# Features

Store temperature data using NumPy arrays

Calculate the average temperature

Identify temperatures above the weekly average

Determine the maximum and minimum temperatures

Display temperatures for each day using loops

# Technologies Used

Python

NumPy

# Purpose of the Project

The goal of this project is to practice using NumPy for numerical data processing while also applying core Python concepts such as loops, conditional statements, and arrays.
